# 🛢 Crude Oil Trading Repository

原油トレードにおける**物理制約テーゼ**の運用リポジトリ。

## 📂 構成

| ファイル | 役割 | 更新頻度 |
|---|---|---|
| [`PLAYBOOK.md`](./PLAYBOOK.md) | 判断フレームワーク、監視指標、ポジション管理ルール | 低（四半期〜半年） |
| [`HEADLINES.md`](./HEADLINES.md) | ヘッドライン時系列ログ | 随時追記 |

## 🎯 コアテーゼ

**「原油市場は政治的ヘッドラインに過剰反応するが、生産・精製・出荷の物理的制約は政治決定では短縮できない。この乖離がエッジ。」**

## 📱 iPhone運用フロー

1. ニュース発見 → **Working Copy** or **GitHubアプリ**で `HEADLINES.md` を開く
2. 最新月のテーブル最上段に1行追記
3. その場でコミット＆プッシュ
4. 週末に PC / Mac mini で振り返り

## 🚦 起動条件

常時監視ではなく、以下のいずれかが揃ったら起動：
- 地政学イベントで1日3%以上の価格変動
- OPEC+計画と実出荷データの明確なギャップ
- 主要産油国のインフラ被害報道
- CFTC建玉の極端な偏り
- カーブ構造の急変

詳細は [PLAYBOOK.md](./PLAYBOOK.md) 参照。

## ⚠️ 運用原則

- AI/半導体ポートフォリオとは**完全に分離**
- イベント駆動・機会主義的、積み上げ型ではない
- テーゼを一文で書けないトレードはしない
- 同テーゼ再追撃禁止
