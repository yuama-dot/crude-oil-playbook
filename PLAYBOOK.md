# 原油トレード プレイブック v1

> **目的**：地政学イベント駆動の原油トレードにおいて、物理的制約とヘッドラインの乖離を機会に変えるための判断フレームワーク。
> **スタイル**：常時監視ではなく、特定条件が揃ったときに起動するイベント駆動型。
> **最終更新**：2026-04-09

---

## 🎯 コアテーゼ

**「原油市場は政治的ヘッドラインに過剰反応するが、生産・精製・出荷の物理的制約は政治決定では短縮できない。この乖離がエッジになる。」**

エンジニアリング視点の応用：
- プラント復旧には長納期部品（大型トランス、特殊ポンプ、制御系）が必要で、調達だけで数ヶ月〜1年
- 電力・水・配管等のユーティリティインフラを無視した増産計画は願望にすぎない
- サプライチェーンの物理時間は政治決定で短縮できない

## 🔍 テーゼ検証フレームワーク

イベント発生時、以下の4象限で整理：

| | ヘッドライン=強気 | ヘッドライン=弱気 |
|---|---|---|
| **物理=強気（タイト）** | 順張り、ただし既にプレミアム乗ってる可能性 | **★最高のエッジ★** 逆張りロング |
| **物理=弱気（緩和）** | **★要警戒★** ショート機会 | 順張り、ただし既に織り込み済み |

→ **目指すのは左下（物理タイトなのにヘッドラインで下げた）を拾うこと**

### 物理的タイトさのチェックリスト

- [ ] **生産能力** 攻撃・事故・設備劣化で物理的に戻らない設備はあるか
- [ ] **電力/ユーティリティ** 変電所・送電線・水処理は復旧したか
- [ ] **長納期部品** 大型トランス、コンプレッサー、カスタムポンプの調達状況
- [ ] **現場人員** 熟練オペレーター・保全要員は戻っているか
- [ ] **精製能力** 製油所稼働率、クラックスプレッド
- [ ] **物流** パイプライン、タンカー手配、港湾機能
- [ ] **在庫** 商業在庫、SPR、フロート在庫（海上）

## 💼 ポジション管理ルール

### サイジング
- **最大枚数**：5枚（前回WTI実績上限）
- **初期エントリー**：2枚（テーゼ強度が明確なとき）
- **追加条件**：事前に価格ではなく「テーゼ強化イベント」で決める
- **口座余力**：常に証拠金の20倍以上を維持

### エントリー
- カタリスト先行認識 → モメンタム確認 → 重要水準安定確認 → エントリー
- 「なんとなく安く見えた」は禁止。テーゼを一文で書けないなら入らない

### エグジット
- **損切り**：テーゼ棄損 or 市場が気づく前に耐えきれないライン
- **部分利確**：レンジ回帰ポイントで1/2
- **残り**：トレーリング（転換線 or 直近安値）
- **完全撤退**：次の地政学カタリストが剥落したら即時

### 禁止事項
- [ ] 同テーゼ再追撃（利確後にすぐ入り直す）
- [ ] 事前に価格をロックインして買い増し宣言
- [ ] ヘッドライン1本で大幅ポジ増
- [ ] 損切りラインのずらし（下方向へ）
